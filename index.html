CHAPTER 01

電學概論

核心公式清單

電流定義

$$I = \frac{Q}{t}$$

I: 電流(A), Q: 電荷(C), t: 時間(s)

電壓與電位

$$V = \frac{W}{Q}$$

V: 電壓(V), W: 功(J), Q: 電荷(C)

電功率 (功率)

$$P = \frac{W}{t} = VI$$

常用導出式：$P = I^2R = \frac{V^2}{R}$

系統效率

$$\eta = \frac{P_{out}}{P_{in}} \times 100\%$$
範例 1.1

若 10 秒內流過導體截面的電量為 20 庫侖，求其電流？

解答：$I = Q/t = 20/10 = \mathbf{2 \text{ A}}$。
CHAPTER 02

電阻

電阻定律

$$R = \rho \frac{L}{A}$$

$\rho$: 電阻率, L: 長度, A: 截面積

溫度與電阻

$$\frac{R_2}{R_1} = \frac{T+t_2}{T+t_1}$$

T: 推算零電阻溫度 (銅為 234.5)

歐姆定律

$$V = IR$$

電導

$$G = \frac{1}{R}$$

單位：西門子 (S) 或 姆歐 ($\mho$)

CHAPTER 03

串並聯電路

串聯總電阻

$$R_T = R_1 + R_2 + \dots$$

並聯總電阻

$$\frac{1}{R_T} = \frac{1}{R_1} + \frac{1}{R_2} + \dots$$

兩電阻並聯：$R_T = \frac{R_1 R_2}{R_1 + R_2}$

分壓法則

$$V_1 = E \cdot \frac{R_1}{R_1 + R_2}$$

分流法則

$$I_1 = I_T \cdot \frac{R_2}{R_1 + R_2}$$

注意：分子是對方的電阻

CHAPTER 04

直流網路分析

戴維寧最大功率轉移定理

條件：當負載電阻 $R_L = R_{th}$ 時

$$P_{max} = \frac{E_{th}^2}{4 R_{th}}$$

節點電壓法 (KCL)： $\sum I_{in} = \sum I_{out}$

迴路電流法 (KVL)： $\sum V = 0$

CHAPTER 05

電容與靜電

電容量定義

$$C = \frac{Q}{V}$$

平行板電容

$$C = \epsilon \frac{A}{d}$$

電場強度

$$E = \frac{V}{d}$$

電能儲存

$$W = \frac{1}{2}CV^2$$

亦等於 $\frac{1}{2}QV$ 或 $\frac{Q^2}{2C}$

CHAPTER 06

電感與電磁

電感量定義

$$L = N \frac{\Phi}{I}$$

或 $L = \frac{N^2}{\mathcal{R}}$

法拉第定律

$$e = L \frac{\Delta i}{\Delta t}$$

感應電勢與電流變化率成正比

互感量

$$M = k \sqrt{L_1 L_2}$$

k: 耦合係數 ($0 \le k \le 1$)

電感儲能

$$W = \frac{1}{2} L I^2$$
